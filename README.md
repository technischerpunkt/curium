# Curium

Quick start guide for [Matrix](https://matrix.org/) with the Element client, written in Vue 3.

## Development

We use [Vite](https://vitejs.dev) as our Bundler.

```
# watch
yarn dev

# production build
yarn build
# preview the production build
yarn preview
```
